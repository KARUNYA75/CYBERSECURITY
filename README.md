🐉 Kali Linux Dual Boot & Linux Fundamentals Documentation  

This repository presents my setup of a Kali Linux dual boot system along with practical exercises on fundamental Linux commands and concepts.

The aim of this project is to demonstrate my hands-on experience with Linux environments, understanding of installation procedures, and command-line proficiency as part of my preparation for cybersecurity-oriented roles.

It includes evidence of the dual boot configuration, terminal activities, and structured notes developed during my learning process.

Contents  
• Dual boot setup proof  
• Boot menu screenshot  
• Terminal command outputs  
• Basic Linux command notes  

System Details  
• OS: Kali Linux  
• Dual Boot: Windows 11  
• Installation: Bootable USB  
• Partition: Manual  

Dual Boot  
The system is configured to run both Kali Linux and Windows 11 which can be selected at startup.
